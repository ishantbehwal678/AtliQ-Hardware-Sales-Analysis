# AtliQ-Hardware-Sales-Analysis

End-to-end data analytics project analyzing multi-year sales data for AtliQ Hardware using SQL and Power BI. Includes data cleaning, revenue validation, business-driven SQL analysis, interactive dashboard development, and insight presentation for executive decision-making.

=================================

1.Project Overview

-This project analyzes sales performance data for AtliQ Hardware, a computer hardware supplier operating across multiple regions in India.

-The objective is to transform fragmented transactional data into structured, reliable insights that support data-driven decision-making through SQL analysis and interactive Power BI dashboards.

2.Dataset

Multi-table relational sales dataset including:

-Transactions (sales amount, quantity, currency, order date, market code)

-Customers

-Products

-Markets

-Date table (year, month attributes)

Data spans across 2019 and 2020, enabling year-over-year revenue comparison.

3.Tools & Technologies

MySQL: Data cleaning, revenue validation, and business-driven SQL analysis
Power BI: Interactive dashboard development and visualization
Excel: Data inspection and validation

4.Work Performed
4.1 Data Cleaning & Validation (SQL)

-Explored transaction data structure
-Joined transaction and date tables for time-based analysis
-Removed invalid transactions (sales_amount ≤ 0)
-Cleaned inconsistent currency entries (INR / USD issues)
-Verified monthly revenue calculations before dashboard creation

4.2 Data Analysis (SQL – MySQL)

Performed business-driven analysis including:
-Year-wise revenue comparison (2019 vs 2020)
-Market-level revenue analysis (e.g., Chennai – Mark001)
-Monthly revenue validation
-Currency standardization checks
-Revenue aggregation for KPI validation

4.3 Data Visualization (Power BI)

Built an interactive executive dashboard including:
-KPI Cards (Total Revenue & Total Sales Quantity)
-Revenue by Market (Regional performance comparison)
-Revenue Trend (Monthly & yearly analysis)
-Top 5 Customers by Revenue
-Top 5 Products by Revenue
-Year & Month slicers for dynamic filtering

4.4 Business Reporting

-Translated technical SQL analysis into business insights
-Highlighted revenue concentration risks
-Identified high-performing markets and customers
-Provided actionable recommendations for leadership

5.Key Insights

-Revenue is concentrated in a few dominant markets (e.g., Delhi NCR).
-A small number of customers contribute a large portion of total revenue.
-Year-over-year comparison supports strategic planning decisions.
-Clean data validation significantly improves dashboard reliability.

6.Outcome

-Delivered an end-to-end sales analytics solution combining SQL and Power BI.
-Transformed raw transactional data into structured insights and an executive dashboard to support performance monitoring and strategic planning.
